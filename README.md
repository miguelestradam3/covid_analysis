# 🦠 COVID-19 Data Analysis using Python

This project performs an exploratory data analysis (EDA) of global COVID-19 confirmed cases and investigates the relationship between the spread of the virus and various socioeconomic indicators from the **World Happiness Report**.

Using **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**, the notebook demonstrates how to clean, merge, analyze, and visualize real-world datasets to uncover meaningful insights.

---

## 📌 Features

- Load and explore COVID-19 datasets
- Aggregate confirmed cases by country
- Visualize infection trends over time
- Calculate daily infection rates
- Identify the maximum infection rate for each country
- Merge COVID-19 data with the World Happiness Report
- Correlation analysis between health, economic, and social indicators
- Scatter plots and regression analysis

## 📊 Datasets

This project combines two publicly available datasets:

### COVID-19 Confirmed Cases

Contains the cumulative number of confirmed COVID-19 cases reported for countries around the world.

### World Happiness Report

Contains several socioeconomic indicators, including:

- GDP per Capita
- Social Support
- Healthy Life Expectancy
- Freedom to Make Life Choices

These indicators are used to explore possible relationships with COVID-19 infection rates.
